# Privacy Boundary

This repo is public. It deliberately excludes the raw evidence layer.

If Eazo/ASI X Inc. prefers that a specific item be handled privately, the public discussion should move out of this repo.

Excluded:

- private screenshots,
- raw chat logs,
- raw backend exception text,
- provider organization IDs,
- request IDs,
- private receipt payloads,
- API keys or pairing tokens,
- local file paths that reveal private infrastructure,
- private agent memory or task ledgers,
- private emails.

Included:

- redacted symptoms,
- product-level findings,
- implementation suggestions,
- public App Store metadata with source URL,
- public bridge design lessons that do not expose secrets.

Where a finding is inferred rather than directly proven, it is labeled as inference.
